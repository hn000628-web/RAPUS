==================================================
SECTION 01 : USERS CORPORATION GRADE
==================================================

users.corporationGrade
= 회사 직급권한 정규 컬럼

권한값:
0 = 수습사원
1 = 팀원
2 = 대리
3 = 팀장
4 = 과장
5 = 상무
6~11 = 확장 직급권한
12 = 최종 OWNER_ACCOUNT / 최고직급권한

DB:
corporationGrade INTEGER NOT NULL DEFAULT 0
CHECK(corporationGrade BETWEEN 0 AND 12)

==================================================
SECTION 02 : LEGACY USER GRADE / GENESIS GRADE
==================================================

users.userGrade
= deprecated legacy account marker

정책:
- 즉시 삭제 금지
- CHECK 범위는 0~12로 유지
- OWNER_ACCOUNT hn0628@naver.com 은 userGrade = 12 적용
- profiles.governanceTier와 혼합 금지

DB:
userGrade INTEGER NOT NULL DEFAULT 0
CHECK(userGrade BETWEEN 0 AND 12)

users.genesisGrade
= 플랫폼 최상위 보안등급 / Genesis 보호 계층

권한값:
0 = NONE
1~11 = RESERVED
12 = GENESIS_OWNER

정책:
- corporationGrade보다 상위 개념
- 일반 운영 capability와 분리
- auth / JWT / payment bypass 금지
- middleware 정상 검증 유지

DB:
genesisGrade INTEGER NOT NULL DEFAULT 0
CHECK(genesisGrade BETWEEN 0 AND 12)
