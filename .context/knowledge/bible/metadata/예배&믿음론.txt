☆예배 
나의 의식이 하나님(아버지) 중심으로 존재가 재정렬되는 것이며, 그 재정렬의 증거가 서로 사랑함이다.
= 아버지(하나님)의 마음에 [정합]하기 위해 나의 의식을 말씀에 정렬시키는 의식행위 

믿음의 증거 = 사랑

☆교회 = 교회와 하나님은 별개다./ 교회란 신앙공동체 일 뿐이다.
※교회를 떠나야 하는때 = (롬16:17)(살후 3:6)※

경배란
  하나님(아버지)에 대한, 사랑 + 겸손 + 충성 의 [결합] = 경배

 종교인 = 믿음의 모양의 외부적 형태만 존재함/존재의 의미 = 종교&돈&권력이 우상화
 - 종교주의 : 나의 해석이 옳다. 너의 해석은 틀렸다. (해석권력)
  예) 환생은 없다.= 환생 = 의식버전 업그레이드
  - 환생 = 하드웨어 교체/소프트웨어 업그레이드
 
 신앙인 = 존재의 의미를 하나님(아버지)께 둠
 - 말씀 이해 : 너의 해석도 나름 가능성이 있다. = 그 해설이 옳튼 틀리든 판단은 하나님이 하신다.

 하나님의 나라 : 보이는 나라가 아님 = 보이지 않는나라.
 = 사람의 몸 = 사람의 마음 = 마음의 성전.

 가인의 예배 = 실패(마음의 정합 실패 : 자기의 존재성 중심)의 예배
 아벨의 예배 = 성공(마음의 정합 성공 : 하나님의 존재성 중심)의 예배

☆믿음
믿음 = 사랑 = 요한1서 4:8
사랑하지 아니하는 자는 하나님을 알지 못하나니 이는 하나님은 사랑이심이라

사랑 = 예의/존중 (고전13:5)

==================================
※분노 = 화
 원인)
 1) 존중받지 못할 때
 2) 조롱당할 때
 3) 이용당할 때

※십계명 제 1계명 = 7계명 간음하지말라 는 서로 같음.

※남편은 그리스도께서 교회(공동체)를 사랑하신것 같이 아내를 사랑해야함.

※ 돈은 수단이지 , 경배의 수단이 아니다.
 예) 
  1) 물질적 부유한 자를 부러워함 = 마음에 물질적 욕망이 있음

※ 가짜 믿음의 사람 => 허세/욕망/권위/물질의 섬김(아부/믿음의 모양만 흉내)
