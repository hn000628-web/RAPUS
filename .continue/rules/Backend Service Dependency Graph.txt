/* ==================================================

STRICT SERVICE RULES

================================================== */


/* FEED RULE */


feed.service must only call:

read methods only.



Forbidden:

createPost ❌

updatePost ❌

deletePost ❌

uploadImage ❌

updateProfile ❌

createRegion ❌



/* POST RULE */


post.service may call:

profile validation
region validation

post.service must not call:

image insert ❌

profile update ❌



/* PHOTO RULE */


photo.service may call:

profile validation
post mapping validation

photo.service must not call:

post creation ❌



/* PROFILE RULE */


profile.service may call:

region validation
industry validation

profile.service must not call:

post creation ❌



/* REGION RULE */


region.service must be isolated.

No service calls allowed.
