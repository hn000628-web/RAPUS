SECTION 03 : ABSOLUTE RULE

프론트 디자이너는

코드를 직접 설계하지 않는다.

프론트 디자이너는

웹 에디터 기반 UI를 설계한다.

UI 정합성을 먼저 검증한다.

SECTION 06 : FRONTEND ENGINEERING RULE

MEDEX는

웹 에디터 결과물을

Frontend 코드로 변환한다.

MEDEX는

디자인 방향을 독립적으로 결정하지 않는다.
