ANIMATION_IP_DESIGN_REFERENCE_GOVERNANCE_SNAPSHOT

SECTION 01 : CORE PRINCIPLE

사용자는 메카닉 애니메이션, 미래형 레이싱, 성장 서사, 팀 서사, 신념 충돌형 세계관을 선호한다.

참고 방향:
- 건담 SEED 계열의 대형 서사 감각
- 사이버포뮬러 계열의 미래형 레이싱 감각
- 청년 성장
- 팀워크
- 기술과 인간의 관계
- 경쟁과 책임
- 존재의 목적

단, 기존 IP의 직접 복제는 금지한다.

SECTION 02 : ALLOWED REFERENCE

허용:
- 장르적 분위기
- 서사 구조
- 감정선
- 메카닉/차량 디자인의 기능적 접근
- 미래 스포츠 연출
- 팀 기반 성장 구조
- 라이벌 구도
- 전쟁/경쟁 속 신념 갈등

SECTION 03 : FORBIDDEN RULE

금지:
- 기존 캐릭터 복제
- 기존 기체/머신 디자인 복제
- 기존 로고/명칭/설정 복제
- 기존 대사/장면 복제
- 특정 작품과 혼동될 정도의 디자인
- 기존 IP의 세계관 차용

SECTION 04 : ORIGINAL IP RULE

모든 만화/웹툰/애니메이션 디자인은 자체 IP화를 전제로 한다.

목표:
- 오리지널 캐릭터
- 오리지널 세계관
- 오리지널 메카닉/머신
- 오리지널 팀 구조
- 오리지널 로고
- 오리지널 서사

SECTION 05 : LONG TERM GOAL

장기 목표:
- 자체 IP 투자
- 자체 웹툰/애니메이션 기획
- 제작사 설립 또는 제작사 협업
- 플랫폼 기반 IP 확장

SECTION 06 : FINAL DECLARATION

기존 명작은 방향성과 감성의 참고 대상이다.

복제 대상이 아니다.

최종 목표는 사용자 고유의 오리지널 IP를 구축하는 것이다.
